# 目標玩家訪談大綱(PRE-002)

> 初稿 2026-07-12。狀態:暫定,待擁有者審;**訪談執行由擁有者進行**(建議來源:自身與師父的模型攝影/收藏社群、IG/Threads 圈、巴哈姆特模型板)。
> 目的:驗證 spec 的三大痛點假設(不知道拍什麼/怎麼搭/怎麼發)與付費意願,做為 MVP 驗證(PRE-010)的質化基礎。

## 受訪者配額(共 8~12 位)

| 類型 | 人數 | 篩選條件 |
|---|---|---|
| 收藏型玩家 | 2~3 | 收藏 ≥20 隻、很少拍照發文 |
| 攝影型玩家 | 2~3 | 每月至少發 1 次模型攝影貼文 |
| 社群經營者 | 2~3 | 模型相關帳號粉絲 ≥500 或穩定經營中 |
| 輕度/新手玩家 | 2~3 | 收藏 <10 隻、入坑 <1 年 |

篩選問卷(報名表 5 題):收藏數量/主要品類(可動、PVC、鋼普拉、黏土人…)/多久拍一次/發佈平台/願意視訊或面訪 30-40 分鐘。

## 訪談腳本(30~40 分鐘)

### 1. 暖場(3 分)
- 最近入手的一隻是什麼?為什麼買它?

### 2. 收藏行為(10 分)
- 你怎麼記錄自己的收藏?(Excel/APP/腦袋/沒有)
- 你有沒有用手機 App 或網站幫收藏「打勾記錄」(像是標記已入手、許願清單那種)?有的話是哪個?為什麼繼續用/後來沒在用?
- 收藏裡有多少比例「開箱後就一直站著」?為什麼?
- 想買清單怎麼管理?錯過預購的經驗?
- **別人(尤其海外觀眾)常常問你「這是哪一隻/哪個牌子」嗎?大概多常遇到?你都怎麼回?**(2026-07-12 新增,驗證資料辨識痛點)

### 3. 拍攝行為與痛點(10 分,核心)
- 上次拍模型是什麼時候?從「想拍」到「按快門」中間發生了什麼?
- **決定「今天拍哪隻、拍什麼主題」要花多久?卡在哪?**(痛點假設 1)
- 有沒有想過讓兩隻不同作品的模型同框?為什麼做/沒做?(痛點假設 2)
- 拍完之後呢?(修圖/發文/存著)寫文案會卡嗎?(痛點假設 3)

### 4. 社群行為(8 分)
- 都看誰的模型攝影?為什麼喜歡?
- 發文時 hashtag 怎麼下?在意互動數嗎?
- **如果收藏社群裡有人把盜版/仿品當正版曬單,你會怎麼看待這個社群?**(2026-07-12 新增,驗證反盜版治理需求)

### 5. 概念測試(8 分)——展示 MVP 畫面(首頁 → 輪盤 → 產生器 → 結果)
- 第一眼覺得這是什麼?會想在什麼時候打開它?
- 「隨機配對 + 場景任務」的結果你會真的照著拍嗎?為什麼?
- 最想要它幫你做哪一件事?最不需要哪個功能?

### 6. 付費意願(4 分)
- 如果免費版每月限 30 次生成,你會…?(無感/夠用/想升級)
- 月費 NT$99~199 的 Pro 你期待多什麼?(不引導,聽開放回答)

### 7. 收尾(1 分)
- 願意當封測名單嗎?可以介紹 1~2 位同好嗎?(滾雪球)

## 綜整方法

- 8 場後做親和圖:痛點出現頻次 ≥5/8 視為驗證成立。
- 概念測試平均 <3 分 → 觸發 spec 20J 對應 Pivot 檢討。
- 產出:`interview-findings.md`(待訪談完成後由紀錄綜整)。
