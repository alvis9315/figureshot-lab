# 圖片匯入說明

由於完整圖片包容量過大，本下載包只包含規格文件與縮圖總覽。

請將你原本的 `figreShotLab參考資料.zip` 解壓後，把圖片依照 `docs/ux/visual-reference-guide.md` 的分類歸檔到：

- `docs/ux/references/landing/`
- `docs/ux/references/character-select/`
- `docs/ux/references/workspace/`
- `docs/ux/references/style/`
- `docs/ux/references/archive/`

Claude Code 任務提示中已要求先完成圖片歸檔，再開始實作。
