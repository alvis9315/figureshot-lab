# MVP 驗證計畫(PRE-010)

> 初稿 2026-07-12。狀態:暫定——**KPI 門檻值為建議值,需擁有者拍板**;指標定義沿用 spec 20F,Pivot 條件對應 spec 20J。

## 驗證目標

用最小成本回答三個問題:①玩家會不會用收藏生成靈感?②生成結果好到願意留下(儲存/分享)嗎?③有沒有付費訊號?

## 測試設計

| 項目 | 內容 |
|---|---|
| 樣本 | 封測 30~50 位(訪談滾雪球 + 社群招募),公開 beta 目標 100 位(= spec Stage 1 成功指標) |
| 週期 | 封測 2 週 + 公開 beta 6 週,共 **8 週** |
| 通路 | 擁有者與師父的社群、IG/Threads、巴哈模型板;分享頁自然回流 |
| 追蹤 | `user_events` 事件(spec 20F.1 已定義清單)+ 每週看板 |

## KPI 門檻(建議值,請拍板)

| 指標 | 定義 | 🟢 通過 | 🟡 觀察 | 🔴 觸發檢討 |
|---|---|---|---|---|
| Activation | 註冊後完成首次生成 | ≥50% | 35~50% | <35% |
| First Gen Completion | 開始輪盤→成功拿到結果 | ≥80% | 65~80% | <65% |
| 收藏建立 | 每活躍用戶平均收藏數 | ≥5 隻 | 2~5 | <2 |
| 生成頻率 | Generations / MAU | ≥8 次 | 4~8 | <4 |
| Save Rate | 生成後儲存比例 | ≥30% | 15~30% | <15% |
| Share Rate | 生成後分享比例 | ≥10% | 5~10% | <5% |
| Regenerate Rate | 重抽比例(品質訊號,過高=結果差) | ≤50% | 50~70% | >70% |
| D7 留存 | 第 7 天回訪 | ≥25% | 15~25% | <15% |
| Caption Copy Rate | 文案被複製比例 | ≥25% | 10~25% | <10% |
| 付費訊號(質化) | 訪談/問卷中「想升級」比例 | ≥30% | 15~30% | <15% |

判讀規則:🔴 任兩項成立 → 啟動對應 Pivot 檢討;🟢 ≥7 項 → 進入 Stage 2(社群期)規劃。

## Pivot 對應表(spec 20J 落地)

| 訊號組合 | 判讀 | 動作 |
|---|---|---|
| 試玩高、收藏建立低 | 建檔門檻太高 | 無帳號快速輪盤 / 可匯入收藏(CSV、MFC) |
| 收藏高、輪盤使用低 | 工具動線弱 | 收藏頁強化「帶進產生器」入口;或轉收藏展示定位 |
| 輪盤高、文案採用低 | AI 產出不合口味 | 降 AI 比重,強化模板與擺拍建議 |
| 分享高、付費訊號弱 | C 端變現難 | 提前佈局品牌挑戰/導購(Stage 4 前移) |

## 週報格式(8 週,每週一次)

```text
第 N 週:新增用戶/活躍/生成數/儲存率/分享率/Top 熱門組合/本週用戶原話 ×3/風險與調整
```
